## 一、核心单元

### KU（Knowledge Unit，知识单元）
一个**原子的、自足的、可陈述的、可关联的**知识单元，是"被表示之物的替身"(surrogate)，不是文本。
- 四属性：原子性(单一连贯事项) / 自足性(脱离原文可理解) / 可陈述性(是断言非问题) / 可关联性(能进网络)。
- 任何 KU **必含概念**(命题由概念搭成)；抽不出概念的不是合格 KU。
- 数量由知识密度决定，不预设。

### 六分类（KU 的 knowledge_type）
| 类型 | 英文 | 本质问题 | 例 |
|---|---|---|---|
| 事实性 | factual | 是什么情况？ | 赤壁之战 208 年 |
| 概念性 | conceptual | X 是什么？什么为真？ | 损失厌恶 |
| 立场性 | positional | 谁主张？(无确定真值，必标 stance_holder) | 凯恩斯 vs 奥地利学派 |
| 程序性 | procedural | 怎么做？ | 待定系数法步骤 |
| 解释性 | explanatory | 为什么？机制如何？(深度来源) | 勾股定理为何成立 |
| 元认知 | metacognitive | 怎么学/思考/反思？ | 间隔重复原理 |

### KC（Knowledge Cluster，知识簇）
一组强相关 KU 的主题聚合 + LLM 簇摘要。**用边(真关系)聚类(Louvain)**，不用概念共指、不用连通分量。理想的 KC = 一个主题的 what+why+how 讲透。是 AII 综合物(标 synthesis_marker，grade 永不最高级)。

### BU（Book Understanding，书级理解）
AII 对一本书的整体把握(论点/立场/核心解释/可信度基调)。最高层综合。含 core_explanations(全书 why)等字段。是 AII 综合物。

## 二、概念与本性（AII 最深的本体层）

### 概念（Concept）
KU 的**构成材料**——KU 是用概念搭成的判断。概念分两层：

- **具体概念（concrete concept）**：落在具体对象/学科上。如 价格弹性、勾股定理、热力学熵。常绑学科。
- **抽象概念（abstract concept）**：本身抽象、跨学科。如 熵、均衡、对称。
- 关系：抽象概念**落到**具体对象 → 具体概念(熵→热力学熵)；具体概念**抽掉对象**→抽象概念。
- **同名+不同学科 = 不同概念**(经济"弹性"≠物理"弹性")，靠 discipline 隔离。
- **同一概念唯一向量**：语义归一后，同一概念全局一条向量(不靠字符串名，靠向量)。

### 相（Appearance / Manifestation）
一个概念**可被观测、描述、测量的面向**——量是多少、对应什么、公式长什么样、在哪个学科显现。**会变、是表象。** 热力学熵、信息熵是"熵"的相。

### 本性（Invariant，本性）★ AII 最珍贵的发现
一个**抽象概念在它所有的相之下，那个不变的内核**。借两个传统精确界定：
- **拓扑不变量(topological invariant)**：咖啡杯↔甜甜圈，形状(相)随意变形，亏格(洞数)不变——不变量就是本性。
- **佛教的"性"**：相是表象(会变)，性是本质(穿透一切相不变)。离相见性。
- **判据(拓扑式)**：把概念跨域搬运(熵：热力学→信息论)，什么**变了**(相)、什么**不变**(本性)？不变的那个就是本性。
- 例：熵的本性 = "无外力则只增、不可逆、有方向"(穿透热力学/信息论都不变)，**不是**"无序的度量"(那是相)。
- **本性是"道"(必然怎么变/不变)，不是"相"(是什么样)。**
- 抽不到则留空——**绝不造假**。
- 英文术语锚定 **invariant**(抽英文书更精确，避免 nature 被理解成"性质/特征"=相)。

### 本性 vs 本性概念（关键区分）
- **某抽象概念的本性**：绑在**单个**抽象概念上(熵的本性、均衡的本性)。是该概念的属性，**还不是本性概念**。
- **本性概念（Invariant Concept）**：**多个**抽象概念**共有**的那个不变量，凝结成的独立实体。**只有被≥2个抽象概念共享，才升格为本性概念。**
- 诞生方式：**涌现**——多个抽象概念的本性向量收敛到一起时，才凝结出本性概念。不预设。
- 同构逻辑：多个具体概念的共性→凝结抽象概念；多个抽象概念的本性的共性→凝结本性概念。判据相同(被下层多成员共有)。

## 三、三种连接（强度/性质不同，绝不互相冒充）

| 连接 | 英文 | 是什么 | 强度 | 媒介 |
|---|---|---|---|---|
| **边** | edge | LLM 识别的 KU 间**真实关系** | 局部强 | explains/causes/... |
| **概念共指** | concept co-reference | 两 KU 含同一概念，可一起检索 | 弱(可发现性) | 同一概念 |
| **本性同源** | invariant-shared | 两概念共享同一本性(不变量)，最深同构 | **最深** | 同一本性 |

- **边**：知识网络的关系骨架，KC 靠它聚。**含同一概念 ≠ 有关系**(共指不是边)。
- **本性同源**：连接**名字不同、学科不搭但本质同构**的概念(经济均衡↔生态稳态)。"见诸相非相"——不被名号/学科的相骗，在不变量层面发现同一。AII 揭示知识最深层联系的机制。

### 受控关系词表（edge.relation_type，10 个，不可自由生成）
explains(解释，why命门) / causes / subsumes / special_case_of / prerequisite_of / contrasts_with / opposes / contradicts / supported_by / same_as(跨源同一→合并不建边)

## 四、认识论维度

### grade（可信度/认识论地位）
KU/边的可信度。**LLM 抽取一律标 unverified(默认)**；提升只能由确证机制写入(grade 铁律)。值：unverified/pending/low/moderate/high/verified/refuted/contradicted。

### grounded_by（可信度来源）
default(未证) / formal_proof(形式验证) / source_crosscheck(来源交叉) / stance_internal(立场内论证，永不verified)。

### 准入闸门（Admission Gate）
什么不配做 KU 的筛选：连接>收集 / 信息vs知识 / 原材料vs成品 / 未来可被需要。**论据/案例→降级为 example 或 supported_by 边，不单独成 KU。**

## 六、一句话总纲

> **KU 是原子知识(六分类)，由概念构成。概念分具体/抽象两层。抽象概念有"相"(表象)与"本性"(invariant，一切相之下的不变内核)。多个抽象概念共享同一本性 → 涌现本性概念 → 本性同源(最深连接)。三种连接(边/概念共指/本性同源)各司其职，绝不互相冒充。一切以"真"为底线——不让"看起来对"冒充"真的对"。**
