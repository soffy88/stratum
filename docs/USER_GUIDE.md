# Stratum 用户指南 (alpha v1.0)

把英文资料消化成自己的知识，5 分钟上手。

---

## Step 1: 注册账号

1. 打开 https://stratum.uex.hk/register
2. 填写用户名、邮箱、密码（密码 ≥ 8 位）
3. 登录成功 → 进入 Stratum 主界面

首次登录会看到引导教程（可随时在「设置 → 重看引导教程」回放）。

---

## Step 2: 上传第一份资料

### 方法 A：上传文件

1. 点击左侧「文档」
2. 将 PDF / EPUB / Markdown 拖拽到上传区，或点击选择文件
3. 右下角 toast 显示「已入库：XXX」 → 进入文档列表

支持格式：PDF、EPUB、Markdown (.md)、纯文本。

### 方法 B：抓取网页

1. 点击「文档」→「输入 URL」
2. 粘贴网页链接（如 `https://en.wikipedia.org/wiki/Kelly_criterion`）
3. 可选填标题、标签、备注，选择「全文」或「摘要」模式
4. 点「抓取并添加到知识库」→ 等待 5-15 秒
5. 显示「✓ 抓取成功」预览 → 点「查看完整 →」进入文档页

### 方法 C：订阅 RSS

1. 点击「文档」→「订阅 RSS」
2. 输入网站首页或 Feed URL（如 `https://news.ycombinator.com/`）
3. 系统自动发现订阅源 → 选择频率（每小时 / 每 6 小时 / 每天）
4. 订阅成功后，新文章自动入库

---

## Step 3: 跑第一个 AI Agent

1. 点击左侧「AI」
2. 选择 **Translation Worker (翻译)**（推荐作为第一个 Agent）
3. 点「运行 Agent」
4. 30-60 秒后显示结果：你的文档已自动翻译为中文

其他常用 Agent：
- **Daily Digest** — 每天总结新入库内容
- **Reading Companion** — 针对你的资料库问答（输入问题）
- **Knowledge Curator** — 整理 inbox 文件并自动分类

---

## Step 4: 体验三层融合检索

1. 点击左侧「搜索」
2. 输入关键词（用中文或英文都可以）
3. 结果来自三层：**你的文档** + **hevi 专业内容** + **你的笔记**
4. 点击结果进入完整阅读

> 提示：搜索支持语义匹配。输入「凯利公式风险控制」能找到包含「Kelly criterion」的英文文档。

---

## Step 5: 进阶功能

### 概念图谱

1. 点击左侧「概念」→ 选择一个概念
2. 点右上角「查看图谱 →」
3. 可拖拽、缩放节点，点击跳转关联文档

### 时光机（历史时间线）

1. 点击左侧「时光机」
2. 按月查看所有入库内容（文档 / 笔记 / 高亮）

### 反向链接（笔记之间互链）

1. 写笔记时用 `[[note_id]]` 引用其他笔记
2. 进入被引用笔记 → 右侧「反向链接」面板显示所有引用来源

### 分享内容

1. 进入笔记详情页 → 点「分享」
2. 获得公开只读链接，可发给他人

## 常见问题

**Q: 上传失败怎么办？**  
检查文件格式是否支持，或文件大小 < 500MB。超大文件可尝试压缩或拆分。

**Q: 网页抓取显示"需要登录访问"？**  
付费墙或需登录的页面无法服务端抓取。建议安装浏览器扩展（roadmap 中），或手动复制核心内容粘贴为笔记。

**Q: RSS 订阅了但没见到新内容？**  
新内容每 N 小时自动检查（按你设置的频率）。也可在「订阅源」页面点「立即拉取」手动触发。

**Q: 搜索结果不准？**  
搜索基于内容的语义向量。内容越完整，搜索越准。可用更具体的关键词重试。
