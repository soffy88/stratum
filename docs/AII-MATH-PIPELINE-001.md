# 数学专门抽取管道设计（AII-MATH-PIPELINE-001）

> 源起：通用管道用经济书标准硬套数学书 → 质量门静默退化、覆盖不全、输出杂乱。
> 决策：按学科做专门管道。数学管道完全适配数学，但继承通用架构与命门。
> 这确立了 AII 的管道模式：**架构与命门通用，抽取实现按学科专门**。

---

## 一、核心原则：通用的 vs 专门的

### 通用（所有学科管道共享，不变）
- **架构**：KU（讲透单元）/ nu（知识粒，带来源组成）/ KC（知识簇）/ BU（书级理解）。
- **命门**：不编（知识忠于书）+ 不漏（完整性校验）+ 不附会（本性）+ 可溯源。
- **方法原则**：讲透（非碎片）；有向关系读出（非 judge）；质量门拦截 + 暂存 + 人工确认；质量门不静默退化（无效要报警）。
- **产出**：和经济书同构的有机体（KU/KC/BU），能在概念层与其他学科对话（导数↔边际本性同一）。

### 专门（数学管道完全适配数学）
数学和经济从根上是两类知识，抽取实现必须专门：

| 维度 | 经济书 | 数学书（专门适配） |
|---|---|---|
| 真知识点 | 核心概念（黑体术语） | **定义 / 定理 / 公式** |
| 公式角色 | 辅助 | **核心知识本身（R6 是命门）** |
| 知识结构 | 概念+论述 | **公理→定义→定理→证明→推论（演绎链）** |
| 讲透什么 | 概念的内涵/外延/用处 | **定理的条件/结论/证明思路/适用/例子** |
| 有向关系 | 因果/推导 | **严格逻辑依赖（定理证明用到哪些前置定义/定理）** |
| 完整性应有清单 | 黑体术语（~15/章） | **定义/定理（Definition/Theorem/定义X.X/定理X.X）** |

---

## 二、数学管道的各步（专门适配）

1. **数学知识点识别**：抽定义/定理/公式/重要例题，不是套黑体或小节名。一章的"应有清单" = 该章所有定义+定理（这是数学的真知识点）。
2. **数学 KU 讲透**：
   - 定理型 KU：条件（前提）/ 结论 / 证明思路 / 适用场景 / 典型例子。
   - 定义型 KU：定义内容 / 直观含义 / 与相关概念的区别 / 用途。
   - 公式型 KU：公式 / 推导 / 适用条件 / 应用。
   - ★公式必须完整保留（R6 命门）——公式残缺则该 KU 不合格，不是标"未恢复"了事。
3. **数学完整性校验**：应有清单 = 定义+定理（细粒度），逐个查覆盖。识别出的知识点数异常少 → 报警（不静默 complete=True）。
4. **数学打磨**：去无关字符（数学的杂乱词，不止"未涉及"，含"未覆盖/未出现/未给出/需查阅"等，或改语义检测）；删空壳。
5. **有向关系（数学逻辑依赖）**：定理依赖哪些前置定义/定理（数学的逻辑链比经济的因果更严格清晰）——读出，非 judge。
6. **KC（按章 + 谱社区）**：按章 KC（数学章节，固定）；谱社区 KC（跨书主题，可与其他学科同主题汇集）。
7. **BU**：数学书的七项（灵魂/定位/根本问题/知识骨架/思维方式/适合谁/诚实边界），忠实校验。
8. **质量门 + 暂存 + 人工确认**：质量门用数学标准；可疑/无效必报警；不入正式库直到人工确认。

---

## 三、★质量门：绝不静默退化（这次失败的核心教训）

通用管道失败 = 质量门检查项硬绑经济书，换数学书静默退化成无效但显示通过。数学管道的质量门必须：

1. **用数学的真知识点标准**：完整性查定义/定理，不是小节名。
2. **自检有效性**：若识别出的知识点数异常少（长章却只识别几个）、或公式大量缺失、或章节数 < 书声称章节数 → **明确报警**，不静默通过。
3. **残留检测不靠固定词表**：语义检测"非覆盖/缺失"表述，或词表覆盖全并持续维护。
4. **回归测试**：用已知烂数据（这次的数学书）测质量门——能抓出覆盖不全/杂乱/章节缺失才算有效。
5. **R6 公式门**：公式是数学命门，公式残缺率超阈值 → 挡在管道外（数学书 R6 要求远高于经济书）。

**原则**：质量门的价值在"诚实报告检查是否有效"，不在"显示通过"。静默退化（假装成功）比明确失败危险。

---

## 四、本性：回到极谨慎（撤销这次的过度宣称）

- 这次"导数↔边际本性同一验通+物化"是过度宣称（单次判定、数学侧仅 8 个薄 KU、无异源复核）。已撤。
- 本性仍为时尚早：要多本书、多次判、异源复核，跨学科不变内核（如变化率）真正稳定浮现时再提。
- "变化率"作为导数↔边际的共同内核，是**合理假设**，不是被验证的本性——不物化、不宣称验通。

---

## 五、管道模式（确立给以后所有学科）

- 数学管道做成后，是 AII 第二个学科管道（第一个是经济/通用）。
- 以后哲学/历史/物理……各做专门管道，都继承通用架构+命门，专门适配该学科的知识形式。
- 共享：架构（KU/KC/BU）、命门、谱社区 KC（跨学科主题）、本性（跨学科同一）。
- 专门：知识点识别、讲透方式、完整性标准、有向关系、质量门标准。

**这样：每个学科管道完全适配自己（不退化），但产出同构有机体（能跨学科对话）。**
