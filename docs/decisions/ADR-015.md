# ADR-015: 单一产品体验 (不分用户群)


**日期**: 对话中段
**状态**: ✅ 已决定

### Wiki 原话
> 你不用考虑哪个为目标, 功能做到极致优秀就行
> 不需要双层设计, 一个产品一个体验, 对所有人

### 决定
- 不为"普通用户"做阉割版
- 不为"高级用户"做技术版
- 内部技术层 (18 medium / substrate / concept 等) **不暴露给用户**
- 但所有用户看到**同一个产品**, 不分级

### 含义
- SPEC v0.5 §3.2 用户不可见的内部概念 (强制 UX 隐藏)
- 但 SPEC §3 schema 完整保留 (内部使用)
