## 份一 · F12 鸿门宴（互见互补 · 必选）{#F12}

**机制卡**：太史公互见法的**互补**形态——同源（史记）异篇非矛盾。与 F2-b（史记篇间**自相矛盾**）合为『同源异篇』两态：矛盾（indep=0，S12/仅记录，locator 必到篇的存在理由）vs 互补（indep=1，拼合，仅记录）。回归网靠这对区分『真冲突』与『侧重差』——抽取器把互见侧重差误报成冲突，即假阳性；把 F2-b 型篇内矛盾拍平，即假阴性。**五裁决未覆盖此机制**。

**Gold 要点**（`F12-hongmen.json`）：
- 一 event `ev:hongmen`（E2）挂三 account：`ac:hongmen-xiangyuji`（项羽本纪·主叙）/ `ac:hongmen-gaozuji`（高祖本纪·刘邦视角简）/ `ac:hongmen-fankuai`（樊郦滕灌列传·樊哙闯帐）。
- 三篇同 `src:shiji` → **同源，独立见证=1（非 3）**。
- 唯一冲突对象 `cf:hongmen-huxian`：dimension=narrative，`presentation_hint=仅记录`——**不是 S12 对勘**（无对立主张，只有侧重差异的显式登记）。
- 主线取项羽本纪，余篇**择要拼合**（非取一弃余）。

**JUDGMENTS 条目**（`JUDGMENTS.md#F12` 全文）：
> **判定：同事异述·互见互补**（一事件 `ev:hongmen`，散见史记三篇）。判定人：**CC**。
> - **★太史公互见法**：鸿门宴散见史记 **项羽本纪 / 高祖本纪 / 樊哙列传**，各有侧重（主叙 / 刘邦视角 / 樊哙闯帐），**互补拼合全貌**。三 account 同 `source_id=src:shiji`、chapter 各异——**同源，独立见证=1**（非3）。
> - **★与 F2-b 的对照（同源异篇两形态）**：F2-b 是史记篇间**自相矛盾**（narrative 冲突，indep=0，示 locator 必到篇）；F12 是史记篇间**互补非矛盾**（互见法常态，indep=1，`presentation_hint=仅记录`）。二者共同界定『同源异篇』的**矛盾 vs 互补**两态——回归网据此区分『真冲突』与『侧重差』。
> - 主线取项羽本纪，余篇择要补入（拼合，非取一弃余、非 S12 对勘）。

**原文引文**（凭记忆备录，见头部核对状态）：
- 项羽本纪：『旦日飨士卒，为击破沛公军！』；『项庄拔剑起舞，项伯亦拔剑起舞，常以身翼蔽沛公，庄不得击。』
- 高祖本纪（简述形态）：『沛公以樊哙、张良故，得解归。』
- 樊郦滕灌列传（侧重樊哙）：『是日微樊哙奔入营诮让项羽，沛公事几殆。』

**抽验问题**：
1. 三篇同源折算 **indep=1（非 3）**——对否？
2. `cf:hongmen-huxian` 判**互补**（仅记录、不上 S12）而 F2-b 判**矛盾**——两态界线立得住否？
3. 主线=项羽本纪 + 余篇拼合（非取一弃余）——对否？

## 份二 · F8 空城计（按语否定改主线）{#F8}

**机制卡**：carrier/按语三态中的**否定态**——裴松之按语（judgment，计 0 见证）不止可质疑（F3）或裁断（F7），还可**驱动 existence→否、主线采否定**。『主线取否定性 judgment 而非任何肯定叙事』是回归网的稀缺样本：抽取器最易犯的错是把单源逸闻当 E1/E2 正面事件入库。**五裁决只覆盖了 F3 的质疑态，未覆盖否定态**。

**Gold 要点**（`F8-kongchengji.json`）：
- `ev:kongchengji`，**evidence_tier=E3**（很可能不实）。
- 两 account：`ac:kongcheng-guochong`（裴注**引书**郭冲《条亮五事》其三——testimony，existence 正面见证 **indep=1**，genre 极低：西晋晚出、对晋宗室夸述）/ `ac:kongcheng-peian`（裴注**按语**——judgment，**计 0**）。
- 裴按**三驳**：时（诸葛亮初屯阳平时司马懿为荆州都督镇宛，不在关中）/ 地 / 事理（举二十万众疑伏兵，正可设防持重，何至便走）。
- **mainline_account_ref=ac:kongcheng-peian**（主线采否定）；郭冲原说保留为 `cf:kongcheng-existence` S12 并陈；number_claim『二十万』署源展示（『宣王率二十万众』）。

**JUDGMENTS 条目**（`JUDGMENTS.md#F8` 全文）：
> **判定：存疑逸闻——就『空城计』存否，郭冲（有）vs 裴松之（驳）。主线采否定。** 判定人：**CC**。
> - 空城计仅郭冲《条亮五事》**单源**（西晋晚出、对晋宗室夸述），`existence 正面见证 indep=1`（genre 极低）；裴松之**按语=judgment**（计 0）以**时（司马懿时镇宛、不在关中）/地/事理**三驳。
> - **主线取否定性 judgment**：判很可能不实、`evidence_tier=E3`；郭冲原说保留为 existence 冲突对象 S12 并陈。这是『主线取按语裁断』而非取某肯定叙事的样本。
> - 后世空城计戏曲母题所本此单源——尤须显式标注非信史（宁缺毋附会）。

**原文引文**（凭记忆备录）：
- 裴注引郭冲三事：『亮乃敕军中皆卧旗息鼓，不得妄出菴幔，又令大开四门，埽地却洒。宣王疑其有伏兵，于是引军北趣山。』
- 裴按（时地驳）：『亮初屯阳平，宣帝尚为荆州都督，镇宛城，至曹真死后，始与亮于关中相抗御耳。』
- 裴按（事理驳）：『就如冲言，宣帝既举二十万众，已知亮兵少力弱，若疑其有伏兵，正可设防持重，何至便走乎？』

**抽验问题**：
1. 折算：郭冲=引书 testimony（indep=1）、裴按=judgment（计 0）——对否？
2. **主线采否定**（mainline=按语，E3）而非『两说并陈不表态』——三态之否定态适用对否？
3. 郭冲说保留为 existence 冲突 S12 并陈（不删不隐）——保留幅度对否？
4. E3 定档（非 E4 传说、非 E2）——对否？

## 份三 · F10 田氏代齐（parent#2 + succession + 似而非同）{#F10}

**机制卡**：parent 组合的**第二实例**（验 F1 结构可复用非孤例）+ 势力 **succession**（姜齐→田齐，同疆易主挂 force 边）+ 『似而非同』处置（相隔 95 年的两支点**不错合**为一事件）。抽取器最易犯：把『田氏代齐』抽成单事件（拍平 95 年过程），或把弑君与立国错合。**五裁决未覆盖 parent/succession**。

**Gold 要点**（`F10-tianshi-daiqi.json`）：
- 过程 `ev:tianshi-daiqi`（E1）⊃ `ev:tianchang-shijian`（田常弑简公·前481）+ `ev:tianhe-liehou`（田和列侯·前386）——与 F1（晋阳之战+命侯）同构。
- 弑简公两 account：`ac:shijian-zuozhuan`（左传·哀公十四年）/ `ac:shijian-shiji`（史记·田敬仲完世家）——**同事异述，indep=2**；田和列侯：`ac:tianhe-shiji` + `ac:tianhe-zztj`（通鉴←史记**派生**，不另计独立见证）。
- **零冲突对象**（conflicts=[]）——本 fixture 测的是结构（parent/似而非同/succession），不是冲突；矩阵 dimension 列记『—（parent/单述）』。
- succession：`fo:jiangqi → fo:tianqi`（force `succession_edges`）；『田常/陈恒』异名在 persons 一次归一。

**JUDGMENTS 条目**（`JUDGMENTS.md#F10` 全文）：
> **判定：parent 组合；两子事件『似而非同』。** 判定人：**CC**。
> - **parent_event 组合#2**（验 F1 结构可复用）：过程 `ev:tianshi-daiqi`（前481–前386）⊃ `ev:tianchang-shijian`（田常弑简公·前481）+ `ev:tianhe-liehou`（田和列侯·前386）。与三家分晋（F1：晋阳之战+命侯）**同构**。
> - **弑简公 vs 列侯 = 似而非同**：D1 差 95 年、D4 骨架异（弑君专政 vs 册命立国）→ 两 event，经 parent 组合于过程，**非合并**（同 F1-b：过程含多事件不错合）。
> - **succession**：`fo:jiangqi → fo:tianqi`（同疆易主，force `succession_edges`）；异名『田常/陈恒』（史记/左传）在 persons 一次归一。
> - 田常弑简公两 account（左传·哀十四 / 史记田敬仲完世家）同事异述；田和列侯 通鉴←史记（派生，不另计独立见证）。

**原文引文**（凭记忆备录；⚠田世家执/弑地名有异文，逐字待核）：
- 左传·哀公十四年：『甲午，齐陈恒弑其君壬于舒州。』（陈恒＝田常）
- 史记·田敬仲完世家：『（田常）执简公于徐州……田常遂杀简公。』
- 史记·田敬仲完世家（田和）：『康公之十九年，田和立为齐侯，列于周室，纪元年。』

**抽验问题**：
1. 弑简公（前481）与田和列侯（前386）判**似而非同**、经 parent 组合而**不错合**——对否？
2. succession 挂 **force 边**（姜齐→田齐）而非事件层合并——对否？
3. 通鉴田和条视为史记**派生**、不另计独立见证——对否？
4. 零冲突对象（本例测结构不测冲突）——成立否？
