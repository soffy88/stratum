- **回归意义**：防止把"三家分晋"这一过程错合成单一事件、或把两支点事件混为一谈。

## F2 · 赵氏孤儿（赵氏之难）{#F2} ★边界案 · Wiki 亲裁

**为何入选**：左传（前583·下宫之难）与史记·赵世家（前597·屠岸贾藏孤）**时间差 14 年、主体集半重合**——恰好压在"同事异述 / 似而非同"边界上。此类边界正是回归网最需要钉死的标准答案，故 gold 判定**由 Wiki 亲裁并记全理由**（CC 仅备齐证据与归因链）。

### 判定 F2-a：左传下宫之难 与 史记·赵世家屠岸贾故事，同一事件否？
**判定：同事异述**（同指"赵氏被诛而赵武复兴"之本事；屠岸贾灭族（existence）与搜孤救孤（narrative）系晚出敷演，落为冲突对象，不另立事件、亦不另立第二次族诛，不并入史实骨干）。
**判定人：Wiki（亲裁·签署 2026-07-21）**。CC 备据、Wiki 亲裁如下：

| 维度 | 左传·成公八年（前583） | 史记·赵世家（前597） | 判定 |
|---|---|---|---|
| D1 时间窗 | 前583 | 前597 | ⚠ 差 14 年，**不齐**——按 §5 D1，仅当 D2–D4 强对齐 + 源系谱支持同述关系时方可归因为 date 冲突 |
| D2 主体集 | 赵武·韩厥·晋景公（+ 赵庄姬） | 赵武·韩厥·晋景公 + **屠岸贾·程婴·公孙杵臼** | ⚠ **半重合**：核心（赵武·韩厥·景公）交，戏剧三人仅赵世家有 |
| D3 地点 | 晋（下宫） | 晋（下宫） | ✅ |
| D4 事理骨架 | 庄姬谮→晋诛赵同赵括→韩厥请立赵武复田 | 屠岸贾私怨→灭赵氏→藏孤替死→复立诛屠岸贾 | ⚠ 骨架**部分歧异**：结局同（赵武复兴、韩厥关键）、过程歧（有无藏孤替死） |

- **裁定归因链（Wiki 亲裁签署 2026-07-21，★首证在先）**：
  0. **首证·史记自裂**（`cf:zhaonan-shiji-internal`）：屠岸贾/搜孤连史记自己的**晋世家**都不载（晋世家因袭左传）——同一部史记内两篇自相矛盾。这是判"赵世家叙事晚出敷演"的**第一条内证**，先于其余归因。
  1. **核心同一锚点**：D3 全同、D2 的**核心** actor（赵武=赵氏孤儿本人、韩厥=复立关键、晋景公=晋君）全交、D4 的**结局骨干**同（赵氏灭而复兴、韩厥请存赵后）——三者指向**同一桩历史本事**。
  2. **差异全部可归因**：D1 的 14 年差 + D4 的藏孤情节 + D2 的戏剧三人，**同源于一个原因**——史记·赵世家采战国赵人追记之**传说增饰**（太史公采异闻），非记录了"另一件事"。源系谱：赵世家戏剧叙事晚出、无近源支撑；左传近源编年可靠。故差异是"同一事件的晚出敷演版本"，**可归因** → 落 §5"同事异述：差异全部可归因并落为冲突对象"。
  3. **为何不判似而非同**：若判似而非同则须"禁合并 + candidate-same 链"，会把"赵氏孤儿=赵武"这一确定同一的人物本事**割裂**为两桩，反而失真；且 spec §10 明列本条为 existence/narrative **冲突**（即预设同一 event 上挂冲突），支持同事异述。
  4. **为何不判同述**：D1/D4 实质歧异、屠岸贾/搜孤 existence 存疑，远未达"全维度对齐且文本重合"，故非同述。
  5. **落地**：主线 = 左传（近源，`mainline_account_ref=ac:zhaonan-zuozhuan`）；赵世家 account **全文保留**（S12 供给）+ `genre_note` 标晚出敷演。差异落**四冲突对象**——① `cf:zhaonan-shiji-internal`（史记篇内分裂·归因链首证，`presentation_hint=仅记录`）；三挂 S12：② `cf:zhaonan-date`（597/583）③ `cf:zhaonan-existence`（屠岸贾灭族存否）④ `cf:zhaonan-narrative-souhu`（搜孤救孤存否）。
- **★调和说否弃（显式在录）**：`rejected_alternatives[reject:zhaonan-tiaohe-two-purges]`——"两次族诛"调和说（谓前597、前583各一次族诛、两说皆真）**否弃**。理由：前597屠岸贾灭族无近源支撑，调和说为消弭14年矛盾**虚构第二次族诛**、增殖无佐证实体（违 parsimony / 宁缺毋附会）。判前597/前583为同一事件之**系年异说**（落 date 冲突），非两桩史实——把系年分歧误当两次真实族诛正是要防的错。
- **命门校验**：判同事异述**不等于合并二叙事**——戏剧情节以 existence + narrative 两冲突显式并陈、各 `independent_witness_count=1`（仅赵世家），**守宁碎片不错合**：不把"晚出敷演"当"史实"并入骨干、亦不虚构第二次族诛。

### 判定 F2-b：史记·赵世家 与 史记·晋世家，同书篇内冲突（★F2-a 归因链首证）
**判定：同源两篇不同述 → 冲突显式记录（`cf:zhaonan-shiji-internal`，independent_witness_count=0）；★兼作 F2-a 归因链首证——屠岸贾/搜孤连史记晋世家都不载，是判其晚出敷演的第一条内证**。判定人：Wiki（亲裁·签署 2026-07-21）。

## F3 · 官渡兵力 + 裴按 {#F3}

### 判定 F3-a：陈寿《三国志》正文 与 裴松之按语，同一事件否？
**判定：同事异述**（同一官渡之战；『兵不满万』与裴松之质疑落为 number 冲突对象）。判定人：**CC**。
- D1 公元200（同）· D2 曹操/袁绍（同）· D3 官渡（同）· D4 以少胜多骨架（同）→ 全对齐，**同一 event 无疑**。分歧仅在**兵力数值/其可信度**，落 `cf:guandu-bingli`（dimension=number）。

### 判定 F3-b：★carrier/original 折算 + judgment vs testimony（smoke test 核心）
**判定：`independent_witness_count=1`（官渡兵数仅陈寿 1 份证词；裴松之按语=judgment 不计）**。判定人：**CC**（语义经核定 2026-07-21：按语是 judgment 非 testimony，不入折算、不动 tier）。
- **对齐证据/归因链**：
  1. **judgment ≠ testimony**：官渡兵数的独立【证词】只有陈寿《三国志》正文 1 份。裴松之按语是对陈寿的【评断】——未提供官渡兵数的独立证词，只据『初起兵五千、青州降卒三十余万』推理质疑。故按语出**冲突对象 + R8 观点归属（归裴松之本人）**，但**不入独立见证折算、不动 E tier**（同 F4-b『两独立说并存 ≠ 升 tier』守门逻辑）。event tier 维持 **E2**（单一近源文献），与 indep=1 一致。
  2. **carrier/original 区分（在此更锋利）**：`src:peizhu`（裴注）是载体，`carrier_of=[魏略, 江表传, 世语]` 是被载**引书**。若裴注**引**某佚籍给出官渡兵数，那是另一份 **testimony**、计入独立见证；本 account 用的是裴松之**按语**（judgment），不计。真正分野不止 carrier vs original，更是**同一 carrier 内『引书(testimony,计)』vs『按语(judgment,不计)』**。
  3. **★反例警戒（smoke test 要防的两错）**：(a) 把裴松之按语当作兵数的第二份独立证词 → indep 多计、E tier 误升；(b) 把『三国志 + 裴注引某书』当一个源 → carrier/original 少计。本 fixture 同时钉住两者。
- **genre 系数**：裴松之给的是推论非官渡实数——质疑成立、但不构成兵数的等重反证；主线仍取陈寿"不满万"，裴按并陈（S12）。**守宁缺毋附会**：不因裴质疑就替曹操编一个兵数。
