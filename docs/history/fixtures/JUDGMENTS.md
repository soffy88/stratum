> **红线**: 本记录为**人工判定**（CC 手工 + F2 由 Wiki 亲裁），**不得由任何待测抽取器/模型生成或验证**（§7 禁 LLM 自产自证）。此即 A 类回归网首批标准答案，防同一性/冲突判定倒退。

## F1 · 三家分晋 {#F1}

**结构**: parent_event `ev:sanjiafenjin`（过程）⊃ `ev:jinyang-zhizhan`（晋阳之战，前455–453）+ `ev:minghou-403`（命侯，前403）。

### 判定 F1-a：晋阳之战的三 account（史记 / 通鉴 / 战国策）是否同一事件？
**判定：同事异述**（一 event 挂多 account，本 spec 中心情形）。判定人：**CC**。

| 维度 | 史记·赵世家 | 通鉴·周纪一 | 战国策·赵策一 | 对齐？ |
|---|---|---|---|---|
| D1 时间窗 | 前455 围 / 前453 灭智 | 前453 灭智 | （策文不系年） | ✅ 对齐（策文缺省不算冲突） |
| D2 主体集 | 智伯·赵襄子·韩·魏 | 同 | 同 | ✅ 核心 actor 全交 |
| D3 地点 | 晋阳 | 晋阳 | 晋阳 | ✅ |
| D4 事理骨架 | 智伯索地→围晋阳灌水→韩魏反戈→灭智三分 | 同骨架 | 同骨架（增策士言辞） | ✅ 骨干同一 |

- **对齐证据**：D1–D4 全对齐；战国策仅在骨架上增策士敷演言辞（dialogue_spans，透传不消费），非事理差异。
- **归因链**：源系谱——通鉴战国部分派生自史记（`sources.json` src:zztj ← src:shiji）；战国策为策士系材料。三者非独立起源却同述，正因同祖源 → **强倾向同事异述**（§5 辅助证据）。差异仅"叙事侧重/言辞详略"，落 `cf:jinyang-independence`（dimension=narrative），非另一事件。
- **判定人**：CC，2026-07-21。

### 判定 F1-b：晋阳之战 与 命侯 是否同一事件？
**判定：似而非同**（不同事件，但同属父过程）。判定人：**CC**。
- D1 前453 vs 前403（差50年，实质不齐、不可归因为同事）；D4 事理骨架不同（军事灭智 vs 册命立国）。→ **两个 event**，经 `parent_event` 组合于 `ev:sanjiafenjin`，**非合并**。这正是 parent_event 机制的用途：**过程含多事件，不把它们错合成一个**。
- **归因链**：满足"任一维度实质不齐且不可归因 → 似而非同 → 禁合并"；以组合结构表达"同属一过程"而非"同一事件"。
- **回归意义**：防止把"三家分晋"这一过程错合成单一事件、或把两支点事件混为一谈。

## F2 · 赵氏孤儿（赵氏之难）{#F2} ★边界案 · Wiki 亲裁

**为何入选**：左传（前583·下宫之难）与史记·赵世家（前597·屠岸贾藏孤）**时间差 14 年、主体集半重合**——恰好压在"同事异述 / 似而非同"边界上。此类边界正是回归网最需要钉死的标准答案，故 gold 判定**由 Wiki 亲裁并记全理由**（CC 仅备齐证据与归因链）。

### 判定 F2-a：左传下宫之难 与 史记·赵世家屠岸贾故事，同一事件否？
**判定：同事异述**（同指"赵氏被诛而赵武复兴"之本事；屠岸贾/程婴藏孤系晚出敷演，落为 existence 冲突对象，不另立事件，亦不并入史实骨干）。
**判定人：Wiki（亲裁）— 待 Wiki 签署**。CC 备据如下：

| 维度 | 左传·成公八年（前583） | 史记·赵世家（前597） | 判定 |
|---|---|---|---|
| D1 时间窗 | 前583 | 前597 | ⚠ 差 14 年，**不齐**——按 §5 D1，仅当 D2–D4 强对齐 + 源系谱支持同述关系时方可归因为 date 冲突 |
| D2 主体集 | 赵武·韩厥·晋景公（+ 赵庄姬） | 赵武·韩厥·晋景公 + **屠岸贾·程婴·公孙杵臼** | ⚠ **半重合**：核心（赵武·韩厥·景公）交，戏剧三人仅赵世家有 |
| D3 地点 | 晋（下宫） | 晋（下宫） | ✅ |
| D4 事理骨架 | 庄姬谮→晋诛赵同赵括→韩厥请立赵武复田 | 屠岸贾私怨→灭赵氏→藏孤替死→复立诛屠岸贾 | ⚠ 骨架**部分歧异**：结局同（赵武复兴、韩厥关键）、过程歧（有无藏孤替死） |

- **裁定归因链（Wiki 待签）**：
  1. **核心同一锚点**：D3 全同、D2 的**核心** actor（赵武=赵氏孤儿本人、韩厥=复立关键、晋景公=晋君）全交、D4 的**结局骨干**同（赵氏灭而复兴、韩厥请存赵后）——三者指向**同一桩历史本事**。
  2. **差异全部可归因**：D1 的 14 年差 + D4 的藏孤情节 + D2 的戏剧三人，**同源于一个原因**——史记·赵世家采战国赵人追记之**传说增饰**（太史公采异闻），非记录了"另一件事"。源系谱：赵世家戏剧叙事晚出、无近源支撑；左传近源编年可靠。故差异是"同一事件的晚出敷演版本"，**可归因** → 落 §5"同事异述：差异全部可归因并落为冲突对象"。
  3. **为何不判似而非同**：若判似而非同则须"禁合并 + candidate-same 链"，会把"赵氏孤儿=赵武"这一确定同一的人物本事**割裂**为两桩，反而失真；且 spec §10 明列本条为 existence/narrative **冲突**（即预设同一 event 上挂冲突），支持同事异述。
  4. **为何不判同述**：D1/D4 实质歧异、屠岸贾情节 existence 存疑，远未达"全维度对齐且文本重合"，故非同述。
  5. **落地**：主线 = 左传（近源，`mainline_account_ref=ac:zhaonan-zuozhuan`）；差异落三冲突对象——`cf:zhaonan-existence`（屠岸贾/程婴，宁碎片不并入骨干）、`cf:zhaonan-date`（前583/前597）、`cf:zhaonan-shiji-internal`（见 F2-b）。屠岸贾故事走 S12 对勘（差异化内容供给，非麻烦）。
- **命门校验**：判同事异述**不等于合并二叙事**——戏剧情节以 existence 冲突显式并陈、`independent_witness_count=1`（仅赵世家），**守宁碎片不错合**：不把"晚出敷演"当"史实"并入骨干。

### 判定 F2-b：史记·赵世家 与 史记·晋世家，同书篇内冲突
**判定：同源两篇不同述 → 冲突显式记录（`cf:zhaonan-shiji-internal`，independent_witness_count=0）**。判定人：Wiki（亲裁）— 待签署。
- **对齐证据/归因链**：史记·晋世家因袭左传（无屠岸贾），史记·赵世家为戏剧叙事——**同一 source（src:shiji）内两篇自相矛盾**。二者源系谱同 → **不构成两重独立见证**（折算 0）。
- **★存在的唯一目的**：证明 `locator` 必到**篇**级（§2.2）。若 locator 只停"史记"书级，`locator.book` 相同、此篇内冲突被抹掉；停到 `locator.chapter`（赵世家 / 晋世家）方可见。**回归意义**：任何改动若使 locator 退回书级，本条冲突消失即为倒退信号。

## F3 · 官渡兵力 + 裴按 {#F3}

### 判定 F3-a：陈寿《三国志》正文 与 裴松之按语，同一事件否？
**判定：同事异述**（同一官渡之战；『兵不满万』与裴松之质疑落为 number 冲突对象）。判定人：**CC**。
- D1 公元200（同）· D2 曹操/袁绍（同）· D3 官渡（同）· D4 以少胜多骨架（同）→ 全对齐，**同一 event 无疑**。分歧仅在**兵力数值/其可信度**，落 `cf:guandu-bingli`（dimension=number）。

### 判定 F3-b：★carrier/original 独立性折算（smoke test 核心）
**判定：`independent_witness_count=2`（陈寿 1 + 裴松之按 1，跨断代独立）**。判定人：**CC**。
- **对齐证据/归因链**：
  1. **carrier/original 区分**：`src:peizhu`（裴注）是**载体**，`carrier_of=[魏略, 江表传, 世语]`——这些是**被载原源**，各自独立登记、独立计独立性（§3.1）。
  2. 本 account `ac:guandu-peizhu-an` 用的是**裴松之【按语】**——裴松之的**原创观点**（R8 观点归属裴松之本人），**不在 `carrier_of` 之列**（非被载源，是载体作者自己说的）。
  3. 故此处独立见证 = 陈寿（西晋，纪传正文，1）+ 裴松之（刘宋，注·按语，1）= **2**，且裴松之明确**质疑**陈寿，是不同断代的独立观察者。
  4. **★反例警戒（smoke test 要防的错）**：若把"《三国志》+ 裴注引《魏略》"当作**一个源说了两遍** → 独立性折算立错（少计）；若把"裴松之按语"混同为"裴注引某佚书" → 观点归属立错（R8 失效）。本 fixture 就是钉住这两处：**carrier ≠ original，按语 ≠ 引书**。
- **genre 系数**：裴松之给的是推论（初起兵五千、青州降卒三十余万），**非官渡实数**——其质疑成立、但不构成官渡兵数的等重反证；主线仍取陈寿"不满万"，裴按并陈（S12）。**守宁缺毋附会**：不因裴质疑就替曹操编一个兵数。

## F4 · 尧舜禅让（E4 传说层）{#F4}

### 判定 F4-a：史记禅让说 与 竹书纪年古本篡夺说，同一事件否？
**判定：同事异述**（同指尧舜权力交接一事；『禅让』与『舜篡尧』落为 narrative 冲突对象）。判定人：**CC**。
- D1 fuzzy（皆无确切纪年，不构成 date 冲突）· D2 尧/舜（同）· D3 传说地望（异说并存，不实质分歧）· D4 骨架：**权力由尧转舜**同，**性质**（授贤禅让 vs 囚尧篡夺）歧 → 差异可归因为"两传统对同一交接的相反评价"，落 `cf:shanrang-narrative`（dimension=narrative）。同一 event。

### 判定 F4-b：★E4 tier 与"两说并存"的关系（回归网要钉死的陷阱）
**判定：`evidence_tier=E4`（传说层），且不因两独立说并存而升级**。判定人：**CC**。
- **归因链**：`cf:shanrang-narrative.independent_witness_count=2`（史记儒家系统 + 竹书魏编年系统，两独立传统）——但**独立见证数 ≠ 史料等级**。两说皆五帝传说层、无考古互证特定事件、成书距事极远 → **E4 恒定**。竹书**今本**（真伪有争）不计入证据、只作参考（§2.3，`src:zhushu-jinben` 排除于 account 外）。
- **★回归意义**：防止"有两个独立文献说 → 自动升 E1"的错折算。E4→生产 R9 **强制 banner**（传说·无确切纪年），无论主线取禅让或篡说，banner 恒亮，**不假装信史**（§6 E4 定义"显式标注，不假装"）。
- **fuzzy 纪年**：`canonical_date.type=fuzzy`（前841 之前不可 canonical 化），屏幕显式标传说、不编精确年份。
