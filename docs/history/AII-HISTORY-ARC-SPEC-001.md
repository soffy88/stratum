## 1. Arc（弧）：跨事件主题簇【D-024】

```
ku:h-arc {arc_id, title, time_window, sub_arcs[]（有序）,
          member_event_refs[], thesis_refs[], narrative_note, status}
```

- 弧 ≠ Stratum 章级 KC（head/body），二者无关，禁混用词。
- 一弧 → 一集或一季；弧是 EpisodePlan 的上游（消费映射 P3 落地，契约届时 bump，D-025）。
- 弧的成员资格判定记录同 fixture 纪律（判定人 + 依据）。

## 2. Thesis（论点）：因果论断的一等对象

```
ku:h-thesis {thesis_id, claim（一句因果论断）,
             attribution ∈ {源内（臣光曰/太史公曰/传中谏语…，带 locator）,
                            后世公版（如读通鉴论，入语料层）,
                            现代考订（自撰摘述+出处登记，原文不入库，D-027）,
                            我方},
             evidence_refs[]（支撑 KU）, counter_refs[]（反证/对立论点）,
             mainline_decision（多论并陈，主线显式选择+理由+判定人）}
```

**双溯源红线（R10，成片级）**：事实句必溯 KU；论断句必溯 thesis 并显示归属；"史"与"论"在成片中可区分。R 审核 agent（P3）的硬门即机械核此条。

## 3. 首弧：晋国衰亡弧（arc:jin-decline，P2 试点）

### 4.3 既有资产接线（P1 必做，不可跳）

### 4.4 抽取器选型协议（解耦，实测定型，不预 pin）

- 候选集：本地量化开源模型（Qwen3 系 8B/14B 量化档起，CC 按在机 VRAM 实测圈定）；云端 API 默认禁（D-026）。
- 评测集：以 25 gold 派生 held-out（防污染：gold 原文不入 prompt），指标 = 事件抽取 P/R + 注册表解析准确率 + 同一性判定一致率 + VRAM/延迟 ≤ 预算（与 Hevi 错峰）。
- 验收闭环：自动抽取 → 注册表解析 → 同一性判定 → 对回归网（A 类）**零倒退** + B 类 recall 报数。抽取候选永不直接成 gold；gold 只出自判定流程（顾问裁）。
