> **实现**：`tools/history/identity_pipeline.py`。两事件描述 → 抽 **D1 时间窗 / D2 主体集 / D3 地点 / D4 事理骨架** → 规则提案**三处置**（同述 / 同事异述 / 似而非同），带证据行；**存疑走似而非同（宁碎片，§5）**；边界案标存疑**送人裁、不自判**。

## 规则（§5 机器化）

- D1：时间窗 ≤5 年重合 / ≤20 年『近』/ >20 年实质不齐；缺省不算冲突。
- D2：核心 actor 交集非空=核心交；两侧非空且不交=空集。
- D3：place 交集。
- D4：event_type 同 + 标题词重合 ≥2 = 同型。
- **提案**：D1 重合 + D2 核心交 + D3 交/缺省 + D4 同型 → **同事异述**；D1 实质不齐 / D2 空集 / D4 异型 → **似而非同**；**其余存疑 → 似而非同（宁碎片）**。
- （同述需文本级重合，管线不判文本层→保守取同事异述；同述留人工/后续。）
