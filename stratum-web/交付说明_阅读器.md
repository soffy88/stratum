# Stratum 前端交付物 — PDF/EPUB 在线阅读器

文档详情页内嵌 PDF/EPUB 阅读器。3 项交付,放进 stratum-web 对应路径。

## 文件清单
```
src/components/DocumentViewer.tsx          新建,PDF/EPUB/其他 三分支阅读器
src/app/(app)/documents/[id]/page.tsx      详情页骨架,original tab 接入 DocumentViewer
package.json                               加 react-pdf / react-reader / epubjs
```

## 安装依赖
```bash
cd stratum-web
pnpm add react-pdf react-reader epubjs
```

## 关键实现点
1. **JWT 鉴权**:file endpoint 需 Bearer token,不能直接 iframe src。用 apiClient fetch 成 blob(responseType:'blob')→ URL.createObjectURL 喂 viewer,卸载时 revokeObjectURL。
2. **动态导入**:DocumentViewer 用 next/dynamic ssr:false(PDF.js/epub.js 仅客户端,服务端渲染会报错)。
3. **PDF.js worker**:CDN 加载,版本跟随 pdfjs.version(已验证 react-pdf v9.2.1 → pdfjs 4.8.69)。
4. **CSS import 路径**:react-pdf v9 是 `dist/esm/Page/*.css`(不是旧版 `dist/Page/*.css`)。已用正确路径。
5. **性能(§6)**:PDF 分批渲染(每批 5 页 + "加载更多"),不一次性全渲染大文件;blob 下载有进度条(onDownloadProgress);缩放工具栏。
6. **四态**:loading(进度%)/ error(下载 fallback)/ 各类型渲染。

## 注意
- 详情页 [id]/page.tsx 是骨架参考。若 stratum-web 已有详情页,把 ★ 标记的 original tab DocumentViewer 接入合并即可,别整体覆盖。
- 移动端:PDF 容器 overflow-auto + max-h-80vh,EPUB react-reader 自适应。

已通过 TypeScript 类型检查(对真实 react-pdf/react-reader 类型,tsc 零错误)+ CSS 路径存在性验证。
